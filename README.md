# Django Ecommerce Store

This is a full-stack Ecommerce Store web application developed using Django.

## Features

✅ User Signup and Login

✅ Secure Authentication System

✅ Product Categories

✅ Product Search Functionality

✅ Product Detail Page

✅ Shopping Cart

✅ Buy Now Feature

✅ Order Placement

✅ Stock Management

✅ Admin Product Management

✅ Responsive User Interface

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS

## Project Purpose

This project was developed to gain practical experience in full-stack web development using Django. It demonstrates user authentication, database operations, CRUD functionality, shopping cart management, and order processing in an ecommerce environment.
